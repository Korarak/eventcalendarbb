# eventcalendarbb
my Eventcalendar

How to install
1. create table on your database like this
  >event_no (auto_increment)
  >txtdetail varchar
  >txtdestination varchar
  >start_date date
  >end_date date
  >date_added date
  >status
2. config your database connection in connect.php
3. start at calendar.php
